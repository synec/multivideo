# Multivideo

This Angular app allow to stream multiple video at once from an HTTP server.

## Install

- `git clone https://github.com/synec/multivideo.git`
- `cd multivideo`
- `npm ci`
- `npm start`

Afterwards app will be accessible at http://localhost:4200
